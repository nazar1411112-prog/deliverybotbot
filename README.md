# Delivery Telegram Bot

Простой и быстрый Telegram-бот для сервиса курьерской доставки в Молдове.

## Описание
Бот автоматизирует процесс заказов. Клиенты могут указывать адреса и оформлять доставку, а курьеры — брать заказы в работу. Встроена поддержка трех языков: русского, румынского (RO) и английского (EN). 

## Главные функции
* Разделение на Клиентов и Курьеров.
* Автоматический расчет расстояния маршрута (через OSRM) и стоимости в леях (MDL).
* Встроенная админ-панель для верификации сотрудников.
* Двусторонняя связь с техподдержкой.

## Установка и запуск

1. Склонируйте репозиторий.
2. Установите нужные библиотеки:
   pip install -r requirements.txt
3. Создайте файл `.env` и добавьте туда токены:
   BOT_TOKEN=ваш_токен
   DATABASE_URL=ссылка_на_базу_данных
   ADMIN_ID=ваш_id
4. Запустите бота:
   python bot.py
